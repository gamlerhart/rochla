About Projekt Rochla

Projekt Rochla gives you a temporary Windows machines within a minute. The machine is ready to use, so you can try out immediately what you want. You have full admin rights on the machine. Go wild!

If you have a Turbo.Net account, Great! Projekt Rochla will fetch your favorites app from turbo.net.

Get Started

The machine will have Chocolatey and Firefox preinstalled for convinience. You'll have full admin rights on this machine. You can install any application you want.

Using Turbo.net?

We can prepare your Turbo.net apps for you. So that their immediately ready to use. You need to provide an API key, which you can find in your Turbo.net Account's Settings. Create a new one for Projekt Rochla. You remoke it to withdraw access. We don't store the API key permanently.
